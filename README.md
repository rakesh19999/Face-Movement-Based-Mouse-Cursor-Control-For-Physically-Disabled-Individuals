# Face-Movement-Based-Mouse-Cursor-Control-For-Physically-Disabled-Individuals
It is a Python project which helps physically disabled people to operate laptop/desktop by their face movements

```

1. imutils 
2. utils 
3. numpy 
4. pyautogui
```
